## 🎯 El problema

> **Falta de opciones de entretenimiento rápido y accesible para sesiones cortas en dispositivos móviles.**

Los juegos móviles actuales suelen requerir:
- ❌ Largas sesiones de juego.
- ❌ Conexión constante a internet.
- ❌ Descargas pesadas y microtransacciones.
- ❌ Curvas de aprendizaje complejas.

---

## 💡 La solución

**Juego arcade que mejora reflejos mientras ofrece experiencia divertida.**

- ✅ Partidas rápidas de pocos minutos.
- ✅ Sin necesidad de internet.
- ✅ Ligero y gratuito.
- ✅ Mecánicas simples pero adictivas.

---

## 🎮 ¿Cómo funciona?

| Paso | Descripción |
|------|-------------|
| 1️⃣ | **Control táctil para mover personaje** - Desliza o toca para mover al personaje horizontalmente. |
| 2️⃣ | **Sistema de colisiones** - Detecta cuando atrapas un regalo (puntos) o chocas con un obstáculo (pierdes vida). |
| 3️⃣ | **Dificultad progresiva** - Los objetos caen más rápido a medida que avanza el juego. |
| 4️⃣ | **Sistema de puntuación y récords** - Guarda tu mejor puntuación localmente. |

## 🛠️ Instalación y uso

### Requisitos previos

- Flutter 3.x
- Dart 3.x
- Dispositivo o emulador Android/iOS

### Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Ezequie1Sc/love_apk.git
cd love_apk

# Obtener dependencias
flutter pub get

# Ejecutar el juego
flutter run
